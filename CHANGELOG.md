# Changelog

All notable changes to this mod will be documented in this file.

## [1.0.0] - 2026-03-26

### Features
- Automatically detects and designates blighted plants for cutting within 2 seconds
- PlantCutting priority boosted to 1 for all colonists during active blight
- Blocks tree cutting, berry harvesting, and sowing during blight so colonists focus on infected crops
- Automatically enables manual work priorities if they were off (restores when blight clears)
- Switches harvest designations to cut on blighted plants (blighted harvest is useless)
- Blocks sowing in blighted areas to prevent new crops from getting infected
- Home zone filter option
- Notification when blight is detected
- All settings toggleable
- 7 languages: English, German, Chinese Simplified, Japanese, Korean, Russian, Spanish

### Fixes
- Fixed colonists prioritizing tree chopping and berry harvesting over blighted crops
- Fixed priority boost not working when manual priorities were disabled
